# future-regret-simulator
Future Regret Simulator (FRS) is an interactive AI-inspired life simulation platform that analyzes habits, discipline, growth, productivity, health, mindset, and decision patterns to predict future regret trajectories.

Built using Next.js, React, TypeScript, and Tailwind CSS, the simulator visualizes behavioral strengths, risk components, burnout indicators, existential dissatisfaction, momentum, and long-term growth paths through dynamic analytics and futuristic UI design.

Features:
• 50+ behavioral analysis inputs
• Real-time regret score simulation
• Growth and risk projections
• Future path visualization
• Behavioral strength analysis
• Burnout and stagnation indicators
• PNG/PDF export support
• Fully responsive futuristic dashboard UI
