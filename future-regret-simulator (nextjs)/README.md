# ⏳ Future Regret Simulator

> **See your future before you live it.** A behavioral projection engine that calculates the compounding effects of your daily habits, discipline, and decisions over a 5-year window.

## 🚀 Overview

The **Future Regret Simulator** asks you to honestly evaluate yourself across multiple life variables. Using a custom nonlinear penalty model, it projects 3 divergent timelines—a Best Case, a Middle Path, and a Drift Path—visualizing exactly how much "Regret Debt" you are accumulating. 

### Core Features
- **Dynamic Decision Engine:** Tweak sliders spanning finances, health, discipline, and risk appetite.
- **5-Year Growth Projections:** See the compounding gap between who you are now and who you could become.
- **Regret Breakdown:** Identifies exactly which life areas are dragging your potential down.
- **100% Private by Design:** All simulations are processed entirely locally on your device. We do not store, send, or use your data.

## 💻 Getting Started

### Prerequisites
Make sure you have Node.js 18+ installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/highmachs/Future-Regret-Simulator.git
   cd Future-Regret-Simulator
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to start the simulation.
